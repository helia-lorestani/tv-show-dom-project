##  TV Show DOM Project 

A dynamic web app to search and explore TV show episodes using the TVMaze API.


## Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6+).

- **Data Fetching:** Axios with Async/Await.


##  Features

- **Episode Listings:** Displays name, image, and summary for all episodes.

- **Auto-Formatting:** Generates episode codes like S01-E05.

- **Live Search:** Real-time, case-insensitive search by name.

- **Episode Selector:** Dropdown menu to jump quickly to any episode.


## How to Run

Open index.html in your browser.


##  Important Notes

- **VPN Required:** Since this project uses the **TVMaze API**, users in certain regions may need to enable a VPN to fetch data and load images correctly.
- **Desktop Only:** This application is currently optimized for **Desktop view** and is not yet responsive for mobile devices.
